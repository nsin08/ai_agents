# FAQ

**Purpose**: Common questions across all levels

---

1. **Do I need a GPU?**
   No. Mock mode works everywhere. Ollama may benefit from GPU but is optional.

2. **Can I skip Beginner?**
   Only if you already understand tools, memory, and basic agent loops.

3. **Which Python version should I use?**
   Python 3.11.

4. **Do labs run on Windows and Linux?**
   Yes. Most labs provide PowerShell and Linux-friendly commands.

5. **How do I enable real LLMs?**
   Set `USE_OLLAMA=true` or use a cloud provider config.

6. **What if a lab fails?**
   Check logs, run tests, and compare with the provided solution.

7. **Are solutions provided?**
   Yes, in workbooks and lab solutions where applicable.

8. **How is progress assessed?**
   Use the assessment rubrics and project deliverables.

9. **Where do I ask questions?**
   Use GitHub Discussions: https://github.com/nsin08/ai_agents/discussions

10. **Do I need to learn LangChain/LangGraph?**
    No. The core is framework-agnostic; frameworks are optional extensions.

11. **How do I manage cost?**
    Use token budgets and model routing.

12. **Can I use a different vector DB?**
    Yes. The interfaces are designed to be pluggable.

13. **Is this suitable for training peers?**
    Yes. The materials are structured for instructor-led or self-study.

14. **How often are resources updated?**
    Quarterly or after major changes.

15. **What is the fastest path to production skills?**
    Complete Beginner + Intermediate, then pick 2 Advanced case studies.
