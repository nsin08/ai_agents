# Assessment Rubrics

**Purpose**: Grading guides for exercises and projects

---

## Rubric Scale

- 4 = Excellent
- 3 = Good
- 2 = Needs Improvement
- 1 = Incomplete

## Exercise Rubric (Chapters)

| Criterion | 4 | 3 | 2 | 1 |
|----------|---|---|---|---|
| Correctness | Fully correct | Minor errors | Partial | Incorrect |
| Explanation | Clear and concise | Mostly clear | Vague | Missing |
| Evidence | Tests or traces | Some evidence | Minimal | None |

## Lab Rubric

| Criterion | 4 | 3 | 2 | 1 |
|----------|---|---|---|---|
| Implementation | Complete | Minor gaps | Partial | Incomplete |
| Reliability | Handles failures | Some handling | Minimal | None |
| Observability | Logs + metrics | Logs only | Minimal | None |

## Project Rubric

| Criterion | 4 | 3 | 2 | 1 |
|----------|---|---|---|---|
| Architecture | Well-structured | Mostly structured | Mixed | Unclear |
| Safety | Clear guardrails | Partial | Minimal | None |
| Performance | Measured + tuned | Measured only | Anecdotal | None |
