# Level 2 Runnable Snippets (Intermediate)

These snippets are short, runnable examples that align with the Intermediate chapters and are validated by tests.

## How to run

From repo root:

```powershell
python curriculum/presentable/02_intermediate/snippets/ch01_orchestrator_state_transitions.py
```

## How to test

```powershell
python -m pytest -q tests/integration/test_intermediate_snippets.py
```
